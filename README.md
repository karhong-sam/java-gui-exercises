# **Introduction to Java GUI**

This repository contains hands-on exercises for both Java Swing and JavaFX

## **Contents**
Ex1: Create a basic frame

Ex2: Create a user handling event (button)

Ex3: Create a basic text box

Ex4: Create a basic checkbox

Ex5: Create a basic menu bar

Ex6: Create a basic file chooser

## **Usage**

All examples are separated into training and solution folders. 
All codes in training folder have few lines commented out so that they can be taught and demonstrated in the class. The solution folder contains the un-commented version for every line of codes.
